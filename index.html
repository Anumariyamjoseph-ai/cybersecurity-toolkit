Cybersecurity Toolkit

Four small tools exploring password security, network recon, tamper detection, and intrusion pattern recognition — try them live below.

01

Password Strength Checker

Checks length, character variety, and common weak passwords — the same logic real strength meters use.

02

Port Scanner

Scans a target for open network ports using nmap. Browsers block real network scans for security reasons, so here's a real sample result from testing this tool on a live server:

PORT STATE SERVICE
22/tcp open ssh
80/tcp open http
8080/tcp open http-proxy
8081/tcp open blackice-icecap
03

File Integrity Checker

Generates a SHA-256 fingerprint of any text you paste. Change even one character and the fingerprint completely changes — this is how tamper detection works.

Fingerprint will appear here.
04

Log Analyzer

Scans login log entries and flags IPs with repeated failed attempts — a classic brute-force attack signature.
